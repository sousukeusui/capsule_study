# CapsuleStudy

![トップページ](./readme1.png)

### アプリ説明
勉強系のWebアプリです。<br>
あらかじめ用意されている問題の中からランダムにテストが作成され、解答することができます。<br>
問題を作成し投稿できたり、逆に様々な問題を解くこともできます。<br>
つぶやくこともでき、勉強についての悩みや目標、情報を発信してみましょう。

### なぜ作ったか
勉強のモチベーションを維持して楽しく継続し、各個人の目標を達成して欲しいという思いから作りました。
